# Retirement Growth Simulator

**Author** Gabe Karpiel

**How To Run** 
1. Compile : 'javac RetirementCalculatorProject.java'
3. Run: 'ava RetirementCalculatorProject'
4. Follow the on-screen prompts

**Assumptions**
- Contributions are split evenly across compounding periods each year
- Contribution increase is applied once per year
- Interest rate and age inputs are capped per the assignment requirements

**Compounding Options Tested**
- Annually
- Monthly
- Daily
